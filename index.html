Circular permutations

Problem. Suppose there are \(5\) identical blue balls and red balls. Compute the number of ways in which all the balls can be arranged in a circular manner.

Solution. We put the red balls in a circular fashion. Since they are identical, there is only one way to perform this. Now we have \(5\) places in which we can put the blue balls. We can put any number of blue balls in each place and the places are distinguishable. A particular arrangement is shown below.

$$ 001011010 $$

The \(0\)'s are the blue balls (there are \(5\) \(0\)'s). There are \(5\) places each separated by a \(1\). Then we just need to find the number of binary sequences of length \(9\) with 5 \(0\)'s and 4 \(1\)'s. This is the same as choosing \(5\) places out of \(9\) distinct places (we will put \(0\)'s in these \(5\) places and \(1\)'s in the remaining ones). The number of ways is accomplish this is:

$$ \binom{9}{5} = \frac{9!}{5!4!} $$
